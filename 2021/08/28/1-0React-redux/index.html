Redux

概念

  • redux是用于javaScrip的状态容器，提供可以预测化的状态管理 。

  • Redux可以让你构建一致化的应用，测试非常方便

  • 还支持界面库，而且体积小2kb

设计初衷

  • 解决混乱的state变化

三大核心

  • 单一数据源

    • 整个应用的state存在一个object tree中

  • state是只读的

    • 唯一改变state的方法就是触发action,action就是一个普通的对象。store.dispatch

  • 使用纯函数来执行修改

redux的组成

  • 2.1 state-状态

就是我们传递的数据，大致分为三类

  • DomainDate :服务器数据

  • UI state : UI的状态

  • APP state: app级别的，全局的

  • 2.2 Action-事件

Component 发送一个action :store

  • Action 的特点

    • 本质是一个javascript的普通对象
    • 内部有一个type属性
    • type定义为字符串

  • 2.3 Reducer
    响应过来的actions,经过处理，把state发送给store的。
    注意： reducer函数必须有return返回值 第一个是state,第二个是action

Component —> reducer —>store

  • 2.4 store

store将action与 reducer联系在一起

  • 主要职责
    • 维护应用的state
    • 提供getState()获取state
    • 提供dispatch()发送action
    • 通过subscribe()来注册监听
    • 通过subscribe()返回值来注销监听

Redux入门案例

  • 构建
    npx create-react-app redux-demo
  • 删除多余的文件
  • 在pages目录创建Home组件 const sendAction= () =>{
    return {
    type :’send_type’,
    value :’ 我是一个action’
    }
    } module.exports ={
    sendAction
    } const initState ={
    value :”默认值”
    } const reducer = (state=initState,action) =>{
    switch(action.type){
    case ‘send_type’:
    return Object.assign({},state,action)
    default:return state;
    }
    } module.exports={
    reducer
    }

import { createStore } from “redux”;

// 导入reducer
import { reducer } from “../reducer”;

// 构建store

const store =createStore(reducer);
// 导出
export default store;

import React from “react”;
import store from “../../store”;
// 导出action构建函数
import { sendAction } from “../../action”;

export default class Home extends React.Component{

handleClick =()=>{
   const action =sendAction(); 
   // 发送一个action ,利用store
   store.dispatch(action);
};
// 当组件加载完毕的时候 ，
componentDidMount(){
    // 注册监听
    store.subscribe(()=>{
        console.log(store.getState())
        this.setState();
    })
};
render(){
    return (
        <div>
          <button onClick={this.handleClick}>click  </button>
          <div>{store.getState().value}</div> 
        </div>
        
    )
}

}
